# Ldap proxy

Ldap proxy for Supermicro BMC LDAP integration with FreeIpa

## How to build for Linux amd64

```
GOOS=linux GOARCH=amd64 go build -o ldapproxy
```

## Alreay builded for Linux amd64 (GO 1.20.5)

```
ldapproxy on this repo
```
